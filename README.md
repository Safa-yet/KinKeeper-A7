# 👥 KeenKeeper — Keep Your Friendships Alive

## 📌 Project Description

**KeenKeeper** is a modern friendship management web app designed to help you stay connected with the people who matter most. It allows users to track interactions, set contact goals, and maintain meaningful relationships through an intuitive and responsive interface.

---

## 🚀 Technologies Used

* ⚛️ React.js
* 🧭 React Router
* 🎨 Tailwind CSS
* 🌼 DaisyUI
* 📊 Recharts
* 🔔 React Toastify
* 📦 JSON (for data storage)

---

## ✨ Key Features

### 1. 📋 Smart Friend Tracking

* View all your friends in a clean card layout
* Track "days since last contact"
* Status indicators: **Overdue / Almost Due / On-Track**

### 2. ⚡ Quick Check-In System

* Instantly log interactions (📞 Call, 💬 Text, 🎥 Video)
* Automatically adds entries to the Timeline
* Real-time toast notifications for actions

### 3. 📜 Interactive Timeline

* View complete history of interactions
* Each entry includes date, type, and title
* Helps you stay consistent with your relationship goals

---

## 🛠️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/keenkeeper.git

# Navigate to project folder
cd keenkeeper

# Install dependencies
npm install

# Run the project
npm run dev
```
